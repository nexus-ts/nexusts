# 33 · Resilience (Retry, Circuit Breaker, Bulkhead)

Three classic distributed-systems primitives — retry with backoff,
circuit breaker, and bulkhead (concurrency limiter) — under a
single, decorator-friendly API.

## What it shows

- `ResilienceModule.forRoot()` — DI singleton that holds default
  config for retry / circuit / bulkhead.
- **`retry()` function** — pure-function retry with backoff
  (constant, linear, exponential, exponential-jitter).
- **`CircuitBreaker` class** — fail-fast circuit with rolling
  failure window, half-open recovery, and shared state across the
  app via `svc.getOrCreateCircuit("stripe")`.
- **`Bulkhead` class** — concurrency limiter with optional FIFO
  queue and `rejectOnFull` for fail-fast.
- **Service-level wiring** — the example uses the resilience
  primitives inline (no method decorators) so the resilience
  layer is visible in the route handler.

The decorators (`@Retry`, `@CircuitBreaker`, `@Bulkhead`,
`@Resilient`) are exported too. They work as metadata-only
decorators in Bun 1.3+'s default stage-3 mode; users on legacy
decorator tsconfig can use `applyResilience()` to wrap a method
manually at controller-mount time.

## How to run

```bash
cd examples/33-resilience-calls
bun main.ts
```

Then in another terminal:

```bash
# First call: 2 internal failures, 1 retry success.
curl http://localhost:3000/retry
# → {"ok":true,"n":3}

# Single call: 1 success.
curl http://localhost:3000/circuit
# → {"ok":true,"n":4}

# Bulkhead: 1 success (concurrency = 2).
curl http://localhost:3000/bulkhead
# → {"ok":true,"n":5}
```

## Code

```ts
import "reflect-metadata";
import { Application, Module, Controller, Get, Inject, Injectable } from "@kabyeon/nexusjs";
import { ResilienceModule, ResilienceService } from "@kabyeon/nexusjs/resilience";

@Injectable()
class FlakyService {
  fetchExternal() {
    n += 1;
    if (n <= 2) throw new Error("transient");
    return { ok: true, n };
  }
}

@Injectable()
@Controller("/")
class AppController {
  constructor(
    @Inject(FlakyService) private flaky: FlakyService,
    @Inject(ResilienceService.TOKEN) private r: ResilienceService,
  ) {}

  @Get("/retry")
  retryRoute() {
    // Plain-function retry. 3 attempts, exponential-jitter.
    return this.r.retry(() => this.flaky.fetchExternal(), {
      attempts: 3,
      initialDelay: 5,
    });
  }

  @Get("/circuit")
  circuitRoute() {
    // Get-or-create a named circuit. Multiple call sites share
    // state — if the upstream flakes, the whole app stops
    // hitting it for `timeout` ms.
    const cb = this.r.getOrCreateCircuit("flaky", {
      threshold: 0.5,   // open when >= 50% of calls fail
      minCalls: 2,      // ...over at least 2 calls
      timeout: 60_000,  // wait 60s before half-open trial
    });
    return cb.execute(() => this.flaky.fetchExternal());
  }

  @Get("/bulkhead")
  bulkheadRoute() {
    // Limit concurrency for the expensive dependency. FIFO
    // queue with a max-queue of 100.
    const bh = this.r.getOrCreateBulkhead("expensive", {
      maxConcurrent: 2,
      maxQueued: 100,
    });
    return bh.execute(() => this.flaky.fetchExternal());
  }
}

@Module({
  imports: [ResilienceModule.forRoot()],
  controllers: [AppController],
  providers: [FlakyService],
})
class AppModule {}

const app = new Application(AppModule);
await app.listen(3000);
```

## When to use which

| Primitive | Use when... |
|-----------|------------|
| **Retry** | The dependency is **expected to flake briefly** (network blips, rate limits, leader election). |
| **Circuit Breaker** | The dependency is **expected to fail for longer periods** (outage, deploy, overload). Don't retry during the storm — back off and let it recover. |
| **Bulkhead** | The dependency is **slow and capacity-limited** (external API rate limit, DB connection pool). Cap the number of concurrent in-flight calls. |

These compose: a typical external call is

```
   Bulkhead (≤ 10 concurrent)
      └─> CircuitBreaker (fail-fast on upstream outage)
            └─> Retry (handle transient blips)
```

`@Resilient({ retry, circuit, bulkhead })` composes all three.
